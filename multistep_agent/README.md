# 🤝 マルチエージェント（ディスカッション型） with Phi-3.5

Google Colab上で3つのAIエージェントがお互いの出力を参照しながらタスクを処理するノートブックです。
登録不要・ビルド不要で動作します。

## マルチエージェントとは

複数の専門エージェントがそれぞれ独立した役割を持ち、お互いの出力を受け取りながら協調してタスクを処理する仕組みです。

```
ユーザーのテーマ
       ↓
┌──────────────────────────────────┐
│  🔍 調査エージェント              │
│  テーマに関する情報を収集する      │
└──────────────┬───────────────────┘
               ↓ 調査結果を渡す
┌──────────────────────────────────┐
│  📊 分析エージェント              │
│  調査結果を受け取り分析・評価する  │
│  → 不足があれば追加調査を依頼     │ ← ディスカッション
└──────────────┬───────────────────┘
               ↓ （追加調査があれば調査エージェントに差し戻し）
┌──────────────────────────────────┐
│  ✍️  執筆エージェント             │
│  調査・分析を統合して最終回答作成  │
└──────────────────────────────────┘
       ↓
    最終回答
```

## マルチステップエージェントとの違い

一見似ていますが、設計思想が根本的に異なります。

| | マルチステップ | マルチエージェント（本ノートブック） |
|---|---|---|
| エージェント数 | 1つ | 3つ（独立したクラス） |
| 処理の担い手 | 1つのエージェントが全フェーズを順番に実行 | 各エージェントが専門領域のみ担当 |
| エージェント間のやりとり | なし | あり（前のエージェントの出力を受け取って処理） |
| 差し戻し | なし | 分析エージェントが調査エージェントに追加調査を依頼できる |
| 拡張方法 | ツールを追加する | エージェントを追加する |
| 適したタスク | ツールを組み合わせた実行処理 | 調査・分析・執筆のように役割分担できるタスク |

## エージェント間の通信

各エージェントはPythonのクラスとして独立して定義されており、**戻り値（文字列・辞書）を介して情報を受け渡します**。HTTPやソケット通信は使用しません。

```python
# 調査エージェントの出力を分析エージェントに渡す例
research_result = research_agent.research(topic)         # 文字列で返す
result          = analysis_agent.analyze(topic,          # 文字列を受け取る
                                         research_result)
analysis_result = result['analysis']
additional_req  = result['additional_request']           # 追加調査依頼（あれば）

# 追加調査依頼がある場合は調査エージェントに差し戻す
if additional_req:
    research_result = research_agent.research(topic, additional_req)
```

### ディスカッションのフロー詳細

```
Round 1:
  🔍 調査エージェント  →  初回調査結果（文字列）
                              ↓
  📊 分析エージェント  →  分析結果 + 追加調査依頼（辞書）
                              ↓ 依頼あり
  🔍 調査エージェント  →  追加調査結果（文字列）

Round 2（必要な場合）:
  📊 分析エージェント  →  再分析 → 追加調査不要 → 終了
                              ↓
  ✍️  執筆エージェント  →  最終回答（文字列）
```

各エージェントは**自分の役割に特化したシステムプロンプト**を持ち、受け取った情報をもとにLLMを呼び出して出力を生成します。LLMの呼び出しはエージェントごとに独立して行われます。

## カスタマイズ

### エージェントの役割を変える

各エージェントの `system_prompt` を書き換えるだけで専門分野を変更できます。

```python
research_agent.system_prompt = "あなたは法律専門のリサーチャーです..."
analysis_agent.system_prompt = "あなたはリスク分析の専門家です..."
writing_agent.system_prompt  = "あなたは法律文書の執筆専門家です..."
```

### エージェントを追加する

```python
class ReviewAgent:
    def __init__(self):
        self.name = '🔎 レビューエージェント'
        self.system_prompt = '最終回答の誤りや漏れを指摘してください。'

    def review(self, topic, final_answer):
        return llm_chat([
            {'role': 'system', 'content': self.system_prompt},
            {'role': 'user',   'content': f'テーマ:{topic}\n回答:{final_answer}'},
        ], max_tokens=300)

review_agent = ReviewAgent()
```

### ディスカッションのラウンド数を変える

```python
run_multi_agent("テーマ", max_discussion_rounds=3)
```
